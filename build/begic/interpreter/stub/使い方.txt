まず、二種類のjarが存在します。
1つ目はbegic.jarです。
ファイルに書かれているソースコードから実行する場合はこちらを使います。
2つ目はibg.jarです。
コマンドラインから逐次実行する場合はこちらを使います。

使えるコマンドは
window width,height
line x1,y1,x2,y2
write "string",x,y // キャンパスに表示
puts "string" // コンソールに表示
の4つのみです。
制御構文はありません。スタブのため、機能が非常に少ないです。

★コマンドプロントまたはターミナルから起動する場合

まずjarファイルのあるディレクトリまで移動します。
------------------------------------------
$ cd build/begic/interpreter/stub
------------------------------------------
次にbegicインタープリタを起動します。
ファイルからの場合です。
------------------------------------------
$ java -jar begic.jar src/BeGIC/stub/test.bg
------------------------------------------
ここでキャンパスが表示されるはずです。
逐次実行の場合です。
------------------------------------------
$ java -jar ibg.jar
iBeGIC>window 400,400
iBeGIC>line 100,100,300,300
iBeGIC>exit
------------------------------------------
400*400のキャンパスに一本の線が浮かぶはずです。

★Javaプログラムから呼び出す場合
jarを通して
(eclipseの場合はプロジェクト配下の右クリックしてプロパティからJavaビルドパスのライブラリからjarを追加してください)
------------------------------------------
import jp.tolz.begic.prototype.stub.Begic;
import jp.tolz.begic.prototype.stub.IBegic;
------------------------------------------
↑のようにインポートをしてください。
利用方法はBegicの場合は文字列の配列を渡します。
------------------------------------------
String[] script = {"window 300,300","line 100,100,100,100"};
new Begic(script);
------------------------------------------
IBegicの場合は一行ずつ文字列を渡します。
------------------------------------------
IBegic ibg = new IBegic();
ibg.eval("window 300,300");
ibg.eval("line 100,100,100,100");
------------------------------------------

バグ等があれば報告お願いします。
