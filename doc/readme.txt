★概要
　BeGIC(仮)プロジェクトは
"Beginner's Graphical Instruction Code"
の省略で「初学者向け視覚的命令型言語」のことである。

★利用対象者
コンピュータの基本的な動作を理解したい者
初等的な物理や数学をプログラミングによって理解したい者
手順や論理の重要性を理解したい者

★BiGIC言語
sampleパッケージにあるようなアプリケーションの作成を行う。
(出展:「増補改訂版 Java言語で学ぶデザインパターン」 結城 浩)
(言語としては全く違うものが用意される)

・インタープリタ
・命令型
・プロシージャ
・グラフィックス
・手続き型

文法はBASIC言語(VBではない初期の純粋なBASICに近いもの)を基準に作成する。
